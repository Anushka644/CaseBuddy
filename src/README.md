# Case Buddy

Case Buddy is an AI-powered PM interview prep tool built to make product management interview preparation more realistic, interactive, and effective.

Most PM interview prep today is passive. Candidates spend time reading frameworks, watching videos, and memorizing answers, but rarely get the chance to practice real product thinking under pressure.

Case Buddy bridges that gap by simulating realistic PM case interviews using AI.

## Problem

Traditional PM prep often lacks active practice. Users know the frameworks, but struggle when they need to think through ambiguous product problems in real time.

The challenge was creating a system that helps users move from passive learning to active decision-making.

## Solution

Case Buddy generates product case prompts, asks interviewer-style follow-up questions, and provides structured feedback on prioritization, tradeoff thinking, communication clarity, and problem-solving approach.

Instead of memorizing answers, users practice how to think like a PM.

## Key Focus

The most interesting part of this project was designing the AI behavior itself.

I focused on deciding when the system should challenge assumptions, ask deeper follow-up questions, guide reflection, and push users to defend their decisions.

This made the project less about interface design and more about designing useful AI behavior inside a product workflow.

## Current Status

The prototype currently runs locally and is being iterated based on testing and feedback.

This project helped me think much more deeply about AI as a product behavior, not just a feature.
